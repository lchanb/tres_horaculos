# Política de Privacidad para "Tres Oráculos Místicos"

**Fecha de última actualización:** 24 de Mayo de 2024

Esta política de privacidad rige el uso de la aplicación de software "Tres Oráculos Místicos" (en adelante, la "Aplicación") para dispositivos móviles.

### ¿Qué información recopila la Aplicación y cómo se utiliza?

**1. Datos proporcionados por el usuario**

La Aplicación no requiere que el usuario se registre o proporcione información personal directamente para su uso básico.

**2. Información recopilada automáticamente**

*   **Entrada de Voz (Permiso de Micrófono):** La Aplicación utiliza la funcionalidad de reconocimiento de voz del dispositivo para capturar las preguntas que el usuario le hace a los oráculos. Este audio se procesa en tiempo real para convertirlo en texto. **La Aplicación no almacena las grabaciones de voz ni las comparte con terceros.** La única finalidad de este permiso es permitir la función de dictado de preguntas.

*   **Datos de Publicidad (Google AdMob):** La Aplicación utiliza Google AdMob como servicio de publicidad. AdMob puede recopilar y utilizar datos anónimos sobre los usuarios para mostrar anuncios personalizados. Esto puede incluir, entre otros, el identificador de publicidad de su dispositivo, la dirección IP (para determinar la ubicación general) y las interacciones con los anuncios. Estos datos ayudan a mostrar anuncios más relevantes y a mantener la Aplicación gratuita. Para obtener más información sobre cómo Google recopila y utiliza sus datos, puede consultar la [Política de Privacidad de Google](https://policies.google.com/privacy).

*   **Datos de Uso y Análisis (Anónimos):** Podemos recopilar información anónima sobre cómo se utiliza la Aplicación (por ejemplo, qué oráculo es el más utilizado, informes de fallos) con el único fin de mejorar la estabilidad, el rendimiento y la experiencia del usuario.

### Permisos requeridos en la Aplicación

*   `android.permission.INTERNET`: Necesario para cargar los anuncios de Google AdMob.
*   `android.permission.RECORD_AUDIO`: Necesario para acceder al micrófono y permitir la función de dictado de preguntas.
*   `NSMicrophoneUsageDescription` (iOS): Justificación para el acceso al micrófono en dispositivos Apple.
*   `NSSpeechRecognitionUsageDescription` (iOS): Justificación para el uso del reconocimiento de voz en dispositivos Apple.

### ¿Compartimos su información con terceros?

No vendemos, intercambiamos ni alquilamos su información de identificación personal a otros. Los datos de voz no se comparten. Únicamente compartimos datos anónimos con nuestros socios de confianza para los fines descritos anteriormente (principalmente, Google AdMob para la publicidad).

### Su consentimiento

Al utilizar la Aplicación, usted acepta nuestra política de privacidad. Si no está de acuerdo con esta política, por favor, no utilice nuestra Aplicación.

Nos reservamos el derecho de actualizar esta política de privacidad en cualquier momento. Le recomendamos que revise esta página periódicamente para estar al tanto de cualquier cambio. La fecha de la última actualización se indicará en la parte superior de esta página.
